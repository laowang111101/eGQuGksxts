# 项目概述 python280

本项目是一款基于Python3.10和Django3.0框架开发的在线考试系统，使用MySQL5.6作为数据库存储。系统旨在为教育行业提供便捷的在线考试解决方案。

## 技术栈

- Python 3.10
- Django 3.0
- MySQL 5.6
- PyCharm（开发工具）

## 功能模块

### 系统登录

- 验证登录用户的身份，根据用户身份进入不同的操作界面。

### 学生管理

- 供管理员使用，维护学生的基本信息。

### 试题管理

- 供教师管理，维护题库内容。

### 组卷

- 教师根据考试科目，从题库中选择符合条件的试题，形成试卷。
- 提供方便的查询功能，帮助教师检索不同要求的试题。

### 在线考试

- 根据学生的班级和登录时间，显示相应的试卷内容。
- 试卷完成后自动评阅，记录成绩，并提供评阅结果。

### 成绩统计

- 按照科目和班级等条件统计学生的考试成绩。

### 成绩查询

- 供教师和学生使用，支持多种查询方式。

## 系统角色

- 学生
- 教师
- 管理员
